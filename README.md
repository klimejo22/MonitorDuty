# Monitor Duty
This tool serves as a shortener of the most usefull commands in lethal company. It is a good QoL addition for anyone who is currently on ship duty and has to write commands over and over

## Instalation
### Pulling the repository
Prerequisites: Having git installed on your device
You need to start your version of the terminal (On windows Win+R then cmd + Enter). After that, paste the following code into the terminal:
```powershell
git clone https://github.com/klimejo22/MonitorDuty.git
```
After that, you have the tool downloaded in the folder that you chose
### Downloading as zip
First, you need to click on the green button labeled "Code" on the top of your screen, then click on the "Download as zip" option, and the download will start

## Running the project
Prerequisites: Having python installed on your device (If you dont have python, either download it from the [official site](https://www.python.org/downloads/) or from Microsoft store)

You just need to run the `main.py` file in python, then the macro will be activated. Beware that it also works outside of the game and will alter the functions of the keys that it binds the commands on. To turn it off, just close the terminal that popped up after you opened it.

## Hotkeys
+ R - view monitor
+ F - switch
+ Q - transmit (you then need to write the message)
+ M - moons
+ 1 - assurance
+ 2 - vow
+ 3 - adamance
